# 华澳数智主页

这是一个可直接发布的静态企业主页，包含基础页面结构、响应式样式和本地主视觉图。

## 项目结构

- `index.html`：页面结构
- `styles.css`：样式与响应式适配
- `assets/hero-platform.png`：本地主视觉图

## 本地预览

直接双击 `index.html` 即可预览。

## 上传华为云

可将整个文件夹内容上传至华为云 OBS 桶，并开启静态网站托管：

1. 上传 `index.html`、`styles.css` 和 `assets/` 文件夹。
2. 在 OBS 桶中开启“静态网站托管”。
3. 首页文档设置为 `index.html`。
4. 如需绑定域名，在域名解析中配置到 OBS 静态网站访问地址。
